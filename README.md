# cf_ai_app
An AI-powered application on Cloudflare.

## Task Planner
- When a user describes a goal or task, the AI agent breaks it into clear, actionable steps and saves it.
- After saving, it gives a brief confirmation listing the steps it created.
- If the user asks to see their goals, it lists them.
- User is able to change the steps, make adjustments and delete steps.
- In the future, turn it into a to-do list.
